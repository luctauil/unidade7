GitHub Pages

É uma configuração disponível no seu projeto que permite colocá-lo no ar e rodar em um link disponibilizado pelo próprio github. É um excelente ambiente para testes.

GitHub Pages

  • Selecione a aba Settings do seu projeto
  • Menu lateral Pages
  • Na tela que abrir haverá a opção Source que solicita a origem do códgo que você quer rodar. Selecione a opção Deploy from a branch.
  • E logo a seguir indique a branch. Geralmente utilizamos a main que á versão mais estável.
  • Aguarde alguns minutos enquanto o GitHub cria seu link e faz o deploy do seu projeto.
  • Atualize a página e veja qual o link da sua aplicação.

Colaboradores

Colaboradores e GitHub Pages

O GitHub permite que você adicione colaboradores ao seu projeto. Observe que isso é diferente de deixar o seu projeto como público. A opção pública deixa o seu projeto vísivel a todos para que possam visualizar, consultar e copiar o seu projeto, mas não podem alterar nada. Somente nesta opção de colaboradores você permitirá que as pessoas alterem e colaborem no seu projeto.

Como adicionar colaboradores?

  • Selecione a aba Settings do seu projeto
  • Menu lateral Collaborators
  • Na página que abrir clique no botão Add people
  • Pesquise pelo login ou e-mail do colaborador.
  • O colaborador receberá um convite por e-mail.
  • Assim que o convite for aceito, ele já conseguirá adicionar ou remover códigos no seu projeto.
